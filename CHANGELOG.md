## [0.0.3] - 2024-01-XX

### 🎉 新增

- ✨ 支持 FIM（Fill-in-the-Middle）模式，智能理解上下文
- 🎯 多选择补全展示（2×2 网格界面）
- ⚡ 防抖优化，停止输入后才触发请求
- 🔄 自动触发：支持删除、换行、空格操作

### 🔧 优化

- 📦 精简依赖，删除无用的包（React、Tailwind 等）
- 🧹 清理代码，删除调试日志和测试文件
- 📝 重写 README 文档，更清晰易懂
- ⚙️ 简化配置项，只保留必要参数

### 🗑️ 删除

- ❌ 移除 AI 聊天功能
- ❌ 移除 Git 提交助手功能
- ❌ 移除所有 UI 组件库依赖

### 🐛 修复

- 🔨 修复插入位置错误的问题
- 🔨 修复防抖逻辑导致的频繁请求
- 🔨 修复补全内容清理逻辑
